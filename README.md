## Installation :arrow_down:

- git clone
- npm start

### Clone

- Clone this repo to your local machine using `https://github.com/metinozkan/E-berber.git`

```shell
$ git clone https://github.com/metinozkan/E-berber.git
```

### Setup

```shell
$ npm install
```

### Start

```shell
$ npm start
```
